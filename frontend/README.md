# AstrumAtlas Frontend

React frontend для каталога астрономических объектов и событий.

## Технологии

- React 18
- React Router
- Tailwind CSS
- Axios
- Lucide React (иконки)
- React Hook Form
- Vite

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте http://localhost:3000

## Структура проекта

```
src/
├── components/
│   ├── Layout/          # Компоненты макета
│   ├── UI/              # UI компоненты
│   └── Forms/           # Формы
├── pages/               # Страницы
├── hooks/               # React хуки
├── api/                 # API клиенты
└── styles/              # Стили
```

## Особенности дизайна

- **Космическая тема**: Темные цвета с градиентами
- **Адаптивность**: Mobile-first подход
- **Современный UI**: Стеклянные эффекты, тени, анимации
- **Доступность**: Семантическая разметка

## Компоненты

### Layout
- `Header` - Навигация и меню пользователя
- `Layout` - Основной макет страниц

### UI
- `ObjectCard` - Карточка астрономического объекта
- `EventCard` - Карточка события
- `LoadingSpinner` - Индикатор загрузки

### Forms
- `LoginForm` - Форма входа
- `RegisterForm` - Форма регистрации

## Страницы

- `/` - Главная страница
- `/objects` - Каталог объектов
- `/events` - Календарь событий
- `/favorites` - Избранные объекты
- `/login` - Вход
- `/register` - Регистрация

## API Integration

Все API запросы проходят через `apiClient` с автоматической обработкой JWT токенов.

## Стилизация

Используется Tailwind CSS с кастомной космической темой:
- Цветовая палитра `space` и `cosmic`
- Градиенты для акцентов
- Стеклянные эффекты для карточек
